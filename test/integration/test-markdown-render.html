## 代码示例 这里是一个 Python 快速生成图片的例子： ```python from PIL import Image, ImageDraw, ImageFont import numpy as np def create_gradient_image(width=800, height=600): """创建渐变背景图片""" # 创建渐变数组 gradient = np.linspace(0, 255, width) gradient = np.tile(gradient, (height, 1)) # 转换为图片 img = Image.fromarray(gradient.astype('uint8')) img = img.convert('RGB') # 添加文字 draw = ImageDraw.Draw(img) draw.text((50, 50), "AI Generated", fill='white') return img # 使用示例 img = create_gradient_image() img.save('output.png') ``` 这段代码展示了如何使用 Python 和 PIL 库快速创建一个带文字的渐变图片。
